CSS (calc) Functions

Calc

In this case we are setting on the child the width: width: calc(50% + 50px), our parent is 400px x 400px, so 50% of that width is 200px. So, we are using the calc function to add that 50%(200px) to 50px, giving our child the 250px width.

Min

In this case we are setting on the child the width: min(200px, 60%), our parent is 400px x 400px, so 60% of that width is 240px. So, 60% is bigger than 200px, so in this scenario the 200px value is being used by our min function

In this case we are setting on the child the width: min(300px, 60%), our parent is 400px x 400px, so 60% of that width is 240px. So, 60% is smaller than 300px, so in this scenario the 60% value is being used by our min function

Max

In this case we are setting on the child the width: max(200px, 60%), our parent is 400px x 400px, so 60% of that width is 240px. So, 60% is bigger than 200px, so in this scenario the 60% value is being used by our max function

In this case we are setting on the child the width: max(300px, 60%), our parent is 400px x 400px, so 60% of that width is 240px. So, 60% is smaller than 300px, so in this scenario the 300px value is being used by our max function

Clamp

In this case we are setting on the child the width: clamp(200px, 40%, 300px), our parent is 400px x 400px, so 40% of that width is 160px. Being smaller than our min value (200px), the 200px value will be used

In this case we are setting on the child the width: clamp(200px, 40%, 300px), our parent is 800px x 400px, so 40% of that width is 320px. Being bigger than our max value (300px), the 300px value will be used

In this case we are setting on the child the width: clamp(200px, 40%, 300px), our parent is 600px x 400px, so 40% of that width is 240px. Being bigger than our min value (200px) and smaller than our max (300px), the 40% value(240px) will be used
